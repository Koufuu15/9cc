# このプログラムについて
このプログラムは、コマンドラインに入力された比較演算、算術演算を**Pythonを使ってアセンブリに**コンパイラするプログラムです。

## 環境構築
このリポジトリをcloneし、その配下でコマンドを開いてください。WSLを推奨します。

## 操作
### 演算
`$ python3 main.py "[式]"`

### コードテストを走らせる
`$ ./test.sh`

# test.shの役割
`test.sh`では、以下の動作を行っています。
1. アセンブリを実行する
2. 出力された値と期待していた値が同じかどうかを比較する
3.
    * 同じであれば最後の行に *OK* と出力する <br>
    * どこかで演算結果が異なれば *23 expected, but got 30* のように出力する
